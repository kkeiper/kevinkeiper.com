Responsive images for RL occur within the following breakpoints:

Desktop - 992 +
Tablet portrait - 768 - 991
Phone landscape - 480 - 767
Phone portrait - up to 479

Fixed images are used specifically within "Hero Images" of specific heights and uses. 

They move only along the X axis and do not scale. Use the appropriate Illustrator template dimensions to help in composing these images.

How to use fixed images:
  1. Set background-postion: cover;
  2. Set background-postion: absolute top 0px;
